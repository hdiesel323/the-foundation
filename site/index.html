Built for Production

Pre-flight Approval Gates

Every workflow starts with a human-reviewed plan. No agent takes action without your explicit approval.

Dependency-based Workflow Engine

Agents coordinate automatically using a directed acyclic graph. Tasks execute in the right order, every time.

Discord Thread Integration

Submit tasks and get updates in Discord. Each workflow gets its own thread for clean, organized communication.

Multi-Model Routing

Claude as primary, Grok for overflow. Automatic failover ensures your workflows never stop.

Critic Chain Review System

Multi-agent review before deployment. Peer agents validate work quality and catch issues early.

Real-time Dashboard

Monitor all workflows, agent activity, and system health from a unified control panel.

100% Open Source

Apache 2.0 license. No vendor lock-in, no hidden costs, no data collection.

Self-host on your own infrastructure. Full control over your agents, data, and API keys.

Get Started in 5 Minutes

1. Clone the Repository

git clone https://github.com/hdiesel323/openclaw-002.git cd openclaw-002

2. Configure Your API Keys

cp .env.example .env # Edit .env with your API keys: # - ANTHROPIC_API_KEY # - XAI_API_KEY (optional) # - DISCORD_TOKEN

3. Start the Platform

docker-compose up -d

4. Open the Dashboard

# Navigate to: http://localhost:3000 # Default credentials: # Username: admin # Password: changeme
